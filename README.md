# Carpeta Public - Assets de Grow Labs

Esta carpeta contiene los recursos estáticos del sistema.

## Imágenes Requeridas

Para que la página de login funcione correctamente, debes colocar las siguientes imágenes en esta carpeta:

### 1. **logogrow.png**
- Logo de Grow Labs
- Recomendación: PNG con fondo transparente
- Tamaño recomendado: 300x300px o mayor (se ajusta automáticamente a altura de 96px)

### 2. **fondogrow.png**
- Imagen de fondo para la página de login
- Recomendación: Imagen de alta calidad (Full HD o superior)
- Tamaño recomendado: 1920x1080px o mayor
- Formato: PNG o JPG

## Estructura de Archivos

```
project/
└── public/
    ├── README.md (este archivo)
    ├── logogrow.png (¡COLOCA EL LOGO AQUÍ!)
    └── fondogrow.png (¡COLOCA EL FONDO AQUÍ!)
```

## Notas

- Los archivos en esta carpeta son accesibles desde la raíz de la aplicación
- Se referencian como `/logogrow.png` y `/fondogrow.png` en el código
- Si las imágenes no se encuentran, se mostrará el espacio vacío o un error en la consola del navegador

## ¿Ya tienes las imágenes?

Simplemente copia y pega `logogrow.png` y `fondogrow.png` directamente en esta carpeta y recarga la aplicación.
